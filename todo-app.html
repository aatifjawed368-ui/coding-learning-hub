Build a Todo App with Local Storage

5 min read Feb 8, 2026

What you'll build

A simple todo app that saves tasks in your browser. No backend needed.

Prerequisites

  • Basic Angular knowledge (components)
  • Angular CLI installed
  • VS Code

Step 1: Create new Angular project

ng new todo-app
cd todo-app
ng serve

Open http://localhost:4200 to see it working.

Step 2: Create Todo Component

ng generate component todo

Edit todo.component.ts:

export class TodoComponent {
  tasks: string[] = [];
  
  addTask(task: string) {
    if (task.trim()) {
      this.tasks.push(task);
      localStorage.setItem('todos', JSON.stringify(this.tasks));
    }
  }
}

Step 3: Add HTML template

Edit todo.component.html:

<input [(ngModel)]="newTask" 
            placeholder="Enter task..." (keyup.enter)="addTask(newTask)">
  <ul>
    <li *ngFor="let task of tasks">{{task}}</li>
  </ul>
